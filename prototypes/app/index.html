List of Projects

get/projects/

Get list of projects the user is authorized to view

Parameters

string
optional
Field to order results by; meaning of label varies based on tool
number
optional
Number of results per page in paginated response
number
optional
Page of results to go to
string
optional
UUID for organization to filter by
string
optional
Only return results that contain these tags
Example Request
curl https://api.rasterfoundry.com/projects/tr_19xi9f2eZvKYlo2CrT9LKC2r \
   -d ordering="createdAt,desc"
